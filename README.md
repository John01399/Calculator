# Calculator App

A simple calculator built with Python and Tkinter.

## Features

- Basic arithmetic operations
- Dark theme UI
- Apple-inspired design
- Backspace support
- Keyboard shortcuts
- Clean project structure

## Project Structure

main.py - Entry point
gui.py - User interface
logic.py - Calculator logic

## Run

```bash
python main.py
